# Calorie Tracker Web App

A web application that allows users to track their daily calorie intake by adding foods and their calorie values.
The application helps users monitor eating habits and maintain a healthier lifestyle.

## Features

* Add food items with calorie values
* View all added food entries
* Calculate total calories consumed
* Delete food entries
* Simple and responsive user interface

## Tech Stack

* Python
* Flask
* SQLite
* HTML
* CSS
